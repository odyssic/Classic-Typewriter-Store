#Online Typewriter Store#

When the vintage comes crashing up against the digital age! 

An online store designed to sell vintage typewriters. I run an etsy store https://www.etsy.com/shop/MachinesForWriters?ref=seller-platform-mcnav and decided to create my own platform to learn Angular. 

Built with assistance from a tutorial by Mosh Hamedani's Angular Course. 

Version: 

Angular 10
Bootstrap 4.5
Deployed to Firebase

Google login functionality. 

1. Clone The repository
2. CD into the repository
3. 'npm install' to update the node packages based on the package.json file
4. 'ng serve' to run the local server.
